# seed-help-desk
Seed Help Desk


## Development Setup

Pre-reqs:

* [Node](https://nodejs.org)
* [Sass](http://sass-lang.com/)

Install:

```sh
$ npm install
$ gulp bower
```

## Styleguide

Install:

```sh
$ npm install
$ gulp styleguide
$ gulp styleguide-serve
```

Then visit [http://localhost:3000](http://localhost:3000)
